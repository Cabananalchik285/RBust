## üõ†Ô∏è How to Use <a name="how-to-use"></a>
After installing RBust, here's how you can start using it:

1. Open the application.
2. Input the URL of the website you want to test.
3. Choose the type of fuzzing you'd like to perform. Options include directory bruteforce and parameter fuzzing.
4. Start the fuzzing process by clicking the "Start" button.
5. Monitor the results displayed in the console window.

This procedure allows you to identify weaknesses or bugs in the web application effectively.
